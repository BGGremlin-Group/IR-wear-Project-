#### Open Source CV Dazzle Patterns
Links to the Official CV Dazzle Project by Adam Harvey: 
https://adam.harvey.studio/cvdazzle – Provides descriptions of various looks and patterns designed to disrupt facial recognition, including research and visuals. 

### GitHub Repo: face-verification-with-CV-dazzle-IOI: 
https://github.com/lovc21/face-verification-with-CV-dazzle-IOI – A project evaluating face verification on obscured images using CV Dazzle techniques, including Python scripts for dazzling functions and testing with DeepFace. 

### GitHub Repo: norecognition: 
https://github.com/hevnsnt/norecognition – Generates adversarial patterns to confuse AI surveillance, inspired by CV Dazzle, with a fuzzer for testing millions of designs. 


### Simplified CV Dazzle Hairstyles: 
https://jaredandrews.com/simplified-cv-dazzle-hairstyles.html – Open patterns using hair and makeup to obscure faces from detection software. 


#### Schematics from Mac Pierce's Camera Shy Hoodie
The Camera Shy Hoodie by Mac Pierce is an open-source DIY project.
All schematics, wiring diagrams, and assembly instructions are available in the official PDF guide:
**Download Link:** https://static1.squarespace.com/static/59fc78f1b07869a18edb2d7b/t/63efd3d9b1343a392f680d7e/1676661721736/CameraShyHoodie_Instructions.pdf 

### Project Page: 
https://www.macpierce.com/the-camera-shy-hoodie - Includes overview, parts list (e.g., IR LEDs, RP2040 microcontroller, batteries), and links to the PDF for detailed schematics. 

**Key details** from the project:
Uses 12 high-powered 940nm IR LEDs in a ring around the hood.
Controlled by an RP2040 microcontroller for pulsing.
Wiring: LEDs connected in series-parallel with current-limiting resistors, powered by rechargeable batteries.
Open-source: Full plans, including circuit diagrams, PCB layouts, and code, are in the PDF for free use and modification.
