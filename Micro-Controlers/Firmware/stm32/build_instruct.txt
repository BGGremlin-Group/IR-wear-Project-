Build Instructions for STM32
Method 1: PlatformIO (Recommended)
1. 
Install PlatformIO: https://platformio.org/install
2. 
Create  platformio.ini  in  firmware/stm32/ :
ini
```
[env:bluepill_f103c8]
platform = ststm32
board = bluepill_f103c8
framework = arduino
upload_protocol = stlink
monitor_speed = 115200
lib_deps = 
    adafruit/Adafruit MPU6050@^2.2.4
    adafruit/Adafruit BusIO@^1.14.1
```    
3. 
Place  stm32_firmware.cpp  in  firmware/stm32/src/ 
4. 
Build:  pio run 
5. 
Export binary:  pio run --target build 
Method 2: Arduino IDE
1. 
Install Arduino IDE
2. 
Add STM32 core: https://github.com/stm32duino/Arduino_Core_STM32
3. 
Select Tools → Board → STM32 Boards → BluePill F103C8
4. 
Select Tools → Upload Method → STLink
5. 
Open  stm32_firmware.cpp 
6. 
Click Sketch → Export Compiled Binary
7. 
File will be  stm32_firmware.bin
